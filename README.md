# 仿支付宝app前端项目

> 基于 Vue2.0 开发，功能还在不断完善中。

## 如何开始

``` bash
# 安装依赖
npm install

# 安装 hips ui
npm i @hips/vue-ui -S --registry=http://nexus.saas.hand-china.com/repository/hzero-npm-group/

# 本地 8080 端口启动服务
npm run dev

# 打包项目
npm run build
```
